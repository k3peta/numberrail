# Number Rail (ナンバーレール)

数字のヒントに従ってグリッド上にレール（線）を引く、ロジックパズルゲームです。全てのマスを埋めて、数字の条件を満たすように線をつなぎましょう。

## 🎮 遊び方

グリッド上の数字セルを始点として、ドラッグして線を引きます。数字には以下の意味があります。

*   **十の位**: 線の**長さ**（始点を含むマスの数）
*   **一の位**: 線が**曲がる回数**

**例: 「31」の場合**
*   長さは **3マス**
*   曲がる回数は **1回**

全てのマスを線で埋め尽くし、かつ全ての数字の条件を満たすとクリアです。

## ✨ 機能

*   **豊富なレベル**:
    *   **チュートリアル (4x4)**: 全13問。基本ルールを学ぶための簡単なステージ。
    *   **Main (6x6)**: 全20問。歯ごたえのある標準ステージ。
    *   **Expert (9x9)**: 全10問。広大なフィールドでの難問ステージ。
*   **チュートリアルアニメーション**: トップページでルールの基本が直感的にわかります。
*   **BGM機能**: 心地よい環境音楽を自動生成（Web Audio API使用）。無音プレイも可能。
*   **タイマーモード**: 3分間の制限時間でクリアを目指すモード。
*   **レスポンシブ対応**: PCのマウス操作だけでなく、スマートフォンのタッチ操作にも対応。

## 🛠️ 技術スタック

*   **Frontend**: React
*   **Build Tool**: Vite
*   **Language**: JavaScript
*   **Styling**: Vanilla CSS (CSS Animations)
*   **Audio**: Web Audio API (No external assets required)
*   **Effects**: canvas-confetti (Clear effects)
